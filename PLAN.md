# 星幕光影工作室 - 商業發展企劃書

## 1. 品牌定位 (Brand Positioning)
星幕光影工作室 (Star Curtain Studio) 是一個結合現代美學與情感敘事的影像空間。
- **核心理念**：透過鏡頭捕捉「星幕」下的微光，詮釋人與環境的情感連結。
- **品牌關鍵字**：光影叙事、現代簡約、情感觸動。

## 2. 業務板塊 (Service Portfolio)
1. **商業攝影 (Commercial)**：針對品牌形象、產品行銷進行精準的光影構築，提升產品溢價。
2. **光影人像 (Portrait)**：不僅是拍照，而是捕捉人物當下的靈魂狀態，強調專業攝影與美學設計的結合。
3. **影像設計規劃 (Visual Design)**：為客戶規劃整體視覺敘事，包含廣告設計與影像行銷策略。
4. **展演空間 (Space)**：作為影視創作與藝術交流的實驗基地。

## 3. 市場策略 (Market Strategy)
- **視覺差異化**：在高雄市場中，以「電影感光影」與「專業商業企劃」作為切入點。
- **數位體驗**：建立專屬網站作為作品集展示窗，優化客戶預約與諮詢流程。

## 4. 營運方針 (Operational Plan)
- **第一階段**：完善視覺識別 (VI) 與作品庫存。
- **第二階段**：深耕高雄本地市場，與在地中小企業、設計品牌建立合作。
- **第三階段**：拓展影像行銷顧問服務，將工作室服務轉型為影像內容製作解決方案。

## 5. 資源配置 (Resources)
- **人員**：負責人主導企劃與攝影，並根據專案彈性外聘視覺設計與剪輯協力。
- **硬體**：專業攝影棚設備、後期剪輯工作站 (即將啟用 Mac Studio M4 Max)。
- **行政**：統一編號 00837665 (高雄仁武)。

## 6. 短期目標 (Action Items)
1. **網站優化**：完成首頁動態設計，增加作品集版塊 (Portfolio)。
2. **數位轉型**：自動化工作流，利用 OpenClaw 處理客戶 inquiries 與專案追蹤。
3. **行銷觸及**：開始於社群媒體發佈光影系列企劃，建立專業口碑。
